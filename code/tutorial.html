Tutorial Python

A linguagem Python é uma das mais simples para iniciantes, além de ser uma das mais promissoras e com maior crescimento no mercado atualmente. Esse tutorial tem o objetivo de apresentar os conceitos básicos da linguagem de forma rápida e testar o conhecimento adquirido no final por meio de um teste, gerando a pontuação do usuário no banco de dados.




Entrada e Saída

Python é uma linguagem de alto nível, o que quer dizer que apresenta uma sintaxe mais próxima da linguagem humana, com vocabulário fácil de ser compreendido apesar de utilizar o inglês como base antes do código passar pelo compilador e ser convertido para linguagem de máquina. Como qualquer outra linguagem, Python possui algumas funções essenciais e palavras reservadas já incluídas em sua biblioteca, facilitando a experiência do usuário em executar o programa. Uma das funções mais importantes que serão usadas em cada código para mostrar um resultado de saída na tela, servindo como forma de "comunicação entre usuário e máquina", é a função print(), onde o usuário pode colocar um ou mais parâmetros entre os parênteses para mostrar o valor de um texto ou variável, além de outras modificações, para serem mostradas como saída do programa.

Classes

Classes são tipos existentes na programação orientada a objeto. Este paradigma determina como as classes funcionam.

Os tipos int, float ou string sao tipos padroes do Python. Uma classe é um tipo criado pelo usuário, em que ele determina suas variáveis internas e seus métodos (funções). A importância das classes esta em definir dados junto à sua forma de manipula-los.
