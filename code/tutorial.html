Entrada e Saída

Python é uma linguagem de alto nível, o que quer dizer que apresenta uma sintaxe mais próxima da linguagem humana, com vocabulário fácil de ser compreendido apesar de utilizar o inglês como base antes do código passar pelo compilador e ser convertido para linguagem de máquina. Como qualquer outra linguagem, Python possui algumas funções essenciais e palavras reservadas já incluídas em sua biblioteca, facilitando a experiência do usuário em executar o programa. Uma das funções mais importantes que serão usadas em cada código para mostrar um resultado de saída na tela, servindo como forma de "comunicação entre usuário e máquina", é a função print(), onde o usuário pode colocar um ou mais parâmetros entre os parênteses para mostrar o valor de um texto ou variável contendo algum tipo de dado, do qual será abordado na seção Variáveis, além de outras modificações, para serem mostradas como saída do programa.

Variáveis

O conceito de variáveis consiste em reservar um espaço na memória e dar a ela um nome, tipo e valor. Os tipos de variáveis mais comuns em Python são int (números inteiros), float (números decimais ou com ponto flutuante), string (caracteres usados entre aspas) e booleano (verdadeiro ou falso). Vale citar que existem funções especiais como int(), float(), str() e bool() para converter tipos de dados para outros quando for possível, além de que é preciso entender que valores entre aspas sempre serão string e não será possível fazer cálculos com valores numéricos em forma de string a menos que ocorra alguma conversão temporária no momento da leitura ou uma sobreposição do tipo da variável antes de utilizar a variável em questão para algum propósito.

Classes

Classes são tipos existentes na programação orientada a objeto. Este paradigma determina como as classes funcionam. Os tipos int, float ou string sao tipos padroes do Python. Uma classe é um tipo criado pelo usuário, em que ele determina suas variáveis internas e seus métodos (funções). A importância das classes esta em definir dados junto à sua forma de manipula-los.
